# Antelio

Welcome to the Antelio project repository. This is the personal portfolio and blog for Elijah Savage, an experienced Product Engineer.

## Features

- **Portfolio Showcase**: View a selection of high-quality product engineering projects.
- **Blog**: Read articles and thoughts on product design, engineering, and more.
- **Interactive 3D Elements**: Enjoy a rich user experience with integrated Spline 3D graphics.
- **Admin Dashboard**: Secure Firebase authentication and an admin interface to manage projects, blog posts, and messages.

## Technology Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion
- **Backend & Database**: Firebase (Authentication, Firestore)
- **3D Graphics**: Spline

## Getting Started

1. Clone the repository.
2. Install dependencies with `npm install`.
3. Set up a Firebase project and add your configuration to the `.env` file (refer to `.env.example`).
4. Run the development server with `npm run dev`.
